= Installation =

Extract the downloaded .zip file and upload the extracted folder to wp-content/plugins/ directory.
Activate the plugin through Plugins Menu in Wordpress.

== Frequently Asked Questions ==

= How to disable "social meta tags" for individual product? =

Add custom field `smeta_disable` and set value `true` for individual product in wp-admin product edit page to disable generating social meta tags for selected product.

= How to overwrite auto generated "social meta tags" for individual product? =

Add any of the following custom fields and set value, that to overwrite social meta tags for selected product.

smeta_og_title
smeta_og_desc
smeta_og_image
